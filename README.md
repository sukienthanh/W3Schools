# W3Schools


 * Clone W3schoos UI and some features like manage lessons, code example, display lesson, login, logout,...

# Prerequirements
*   .NET Core SDK
*   SQL Server

# Technologies: 
*   .Net Core 5.0 
*   Dapper ORM
*   Javascript
*   Jquery
*   HTML
*   CSS
*   DevExtreme
*   SQL Server
*   Using JWT for validate client with REST API.

* Open solution in Visual Studio
